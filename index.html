What is this?

This tool generates the actions-nsg-deployment.bicep file required when configuring Azure private networking for GitHub-hosted runners. The generated file contains the Network Security Group (NSG) rules needed to allow outbound connectivity from your Azure VNET.

Data Residency (GHE.com)

If you use GitHub Enterprise Cloud with data residency, your Bicep file requires additional IP ranges specific to your region. This tool automatically adds the correct Actions IPs and regional ingress IPs to the AllowOutBoundGitHub section of the NSG rules. See Network details for GHE.com for more information.

How to Use

1. Select your platform — GitHub.com or GHE.com (data residency).
2. If using GHE.com, choose your data residency region.
3. Click Generate & Download to download the actions-nsg-deployment.bicep file.
4. Use the file with the Azure CLI deployment script as described in the prerequisites documentation.
